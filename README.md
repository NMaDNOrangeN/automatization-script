# 🐍 Лаунчер сборки (Python GUI)

Графическая версия универсального лаунчера для портативных инструментов, написанная на Python 3 и Tkinter. Полностью заменяет консольный BAT-скрипт, предоставляя современный интерфейс, расширенные возможности и удобное управление коллекцией программ.  
**Эта ветка содержит только Python-код** и предназначена для дальнейшей разработки, модификации и сборки в автономный `.exe`.

## 📋 Основные возможности

- **Современный GUI** на `tkinter` с вкладками «Запуск», «Редактор», «Отчёт».
- **Два режима запуска**:
  - *Последовательно* – каждая программа ждёт завершения предыдущей.
  - *Параллельно* – все выбранные программы стартуют одновременно в фоновых процессах.
- **Поддержка типов инструментов**:
  - `gui` – окно не создаётся, вывод перенаправляется в лог.
  - `console` – открывается интерактивное окно `cmd` в папке программы, либо выполняется с произвольными аргументами (фоновый режим с сохранением лога).
- **Гибкий редактор инструментов**:
  - Добавление, изменение, удаление записей.
  - Задание имени, типа, относительного/абсолютного пути и описания.
- **Корневой путь** – базовая директория, относительно которой разрешаются пути инструментов (идеально для переносных сборок на флешке).
- **Умные логи** – каждый запуск записывается в `launch_log.csv` (потокобезопасная запись).
- **Отчёты с очисткой** – просмотр статистики, сохранение отчёта в выбранное место с последующим удалением всех временных логов и CSV-журнала (опционально).
- **Сборка в один exe-файл** с помощью PyInstaller (режим `--windowed`).
